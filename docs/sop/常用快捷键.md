# 个人常用快捷键

## IDEA快捷键

| 快捷键               | 功能                                                         |
| -------------------- | ------------------------------------------------------------ |
| `Alt+S`              | 开关vim                                                      |
| `Alt+/`              | 代码提示补全                                                 |
| `Alt+Enter`          | 导入包，**自动修正代码**(重中之重，需要光标在红色代码行)；实现或重写父类/接口方法 |
| `Ctrl+Y`             | 删除光标所在行                                               |
| `Ctrl+D`             | 复制光标所在行的内容，插入光标位置下面                       |
| `Ctrl+Shift+Alt+L`   | 格式化代码                                                   |
| `Ctrl+/`             | 单行注释                                                     |
| `Ctrl+Shift+/`       | 选中代码注释，多行注释，再按取消注释                         |
| `Alt+Shift+上下箭头` | 移动当前代码行                                               |

文档注释: /**+Enter

按住Ctrl键不放，鼠标点击标识符，可跳转到定义处(或者Ctrl+B)

Alt+Insert 生成JavaBean

Ctrl+Alt+M 将选中的代码抽出来封装为一个方法

new Classname().var->快速生成对象

arr.fori -> 快速生成for循环

Ctrl+N: 搜索类、文件、符号、操作
Alt+7: 查看结构(查看类的方法)

Ctrl+Alt+<-: 跳到上一个位置
Ctrl+Alt+->: 跳到下一个位置



Ctrl+Shift+U，将选中的代码大小写转换

Ctrl+Alt+T, 包围所选代码块

Ctrl+E，最近的文件

Ctrl+Alt+M, 封装代码成方法

控制台选择启动项，Ctrl+D：拷贝启动项

Ctrl+H, 查看类的层级

Ctrl+P, 查看方法的参数

## Linux快捷键



| 快捷键   | 功能                                   |
| -------- | -------------------------------------- |
| `Ctrl+Z` | 挂起作业                               |
| `jobs`   | 查看挂起的作业                         |
| `fg`     | 将后台的作业切换到前台(默认最近的作业) |

```Bash
## Ctrl+Z挂起作业
## jobs查看挂起的作业
## fg 将后台的作业切换到前台

$ jobs
[1]+  Stopped                 vim Git学习笔记.md

$ fg 1
vim Git学习笔记.md

[1]+  Stopped                 vim Git学习笔记.md

Shinelon@DESKTOP-69E8U3H MINGW64 /e/坚果云/Notes/Git
$ fg
vim Git学习笔记.md
```

## VSCode快捷键

### 1. 基本功能

- `Ctrl+N`新建文件

- 在Normal模式再次点击`Esc`**禁用Vim,** 再次点击`Esc`**启用Vim**

  

  ```bash
  # 在keybindings.json文件中添加: 
  	{
          "key": "escape",
          "command": "toggleVim",
          "when":"editorTextFocus && vim.mode == 'Disabled' || editorTextFocus && vim.mode == 'Normal'"
      },
  ```

  

- `Ctrl+Shift+P`打开命令面板

- `Ctrl+Shift+F`全局搜索

- `Create Table of Contents`在命令面板里搜索该命令创建目录，`update`关键字更新目录

- 在命令面板搜索关键字`section`可查找目录编号和更新编号功能

- `Ctrl+B`关闭左边任务栏（光标不在文本中，可以通过鼠标右键）

- `Ctrl+Alt+I`打开Copilot聊天 

- `Ctrl+Shift+I`打开Copilot Edits

- `Alt+<-`往前跳转

- `Alt+->`往后跳转

### 2. Markdown使用

- `Ctrl+K V`分屏预览Markdown文件（退出vim，松开Ctrl+K后按V）
- `Ctrl+Shift+V`新窗口预览Markdown文件



##  Zotero快捷键

- `Shift+P`打开命令栏
- `Alt+<-`往前跳转
- `Alt+->`往后跳转

## Edge

### 沉浸式翻译

搜索框`Alt+I`/空格三连击，翻译搜索框内容

### CLI

#### Windows Terminal

- `Ctrl+Shift+T`打开新标签页

- `Ctrl+Shift+W`关闭标签页
- `Tab`补全

#### Codex

- `Shift+Tab`切换计划/执行模式

- `Ctrl/Shift+Enter`换行

- `Ctrl+G`调起编辑器

- `> codex resume [--list]`

- `/status`查看状态（模型、会话长度、额度使用情况等）

- `->`补全

- 双击`Esc`回滚，只能回滚会话，需要通过git实现真正的回滚

- `@<file_name>`上传文件（只能在当前会话文件夹）

- `Ctrl+U`整行清空（清空光标左边）

- `$<skill_name>`手动使用skill

  

#### Claude Code

- `Ctrl+Enter`换行
- `Ctrl+G`调起编辑器
- `Ctrl+B`后台挂起任务，`/tasks`查看
- 双击`Esc`回滚
- 上传文件：拖拽，`Ctr+C/V`
- `> claude -c`自动恢复上一次会话
- `> claude resume`  和 `/resume`
- `/<skill_name>`手动使用skill
